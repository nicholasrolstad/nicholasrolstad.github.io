An Interactive Geologic Map

This an interactive geologic map created using the Leaflet.JS library and HTML5, CSS & JavaScript. The data used is from the Utah Geologic Survey and was processed in ArcMAP using custom-built tools created with ArcPy. Some of the code used in the processing of data and the live web-map is available by following the link to my GitHub page below. Clicking on the screenshots below will take you to the live web-map (this is a data-heavy map that can take a few seconds to load, not mobile-friendly). This project won First Place in the MAGIP Student Poster Contest at the 2017 Montana State University Earth Science Colloquium in Bozeman, MT and was accepted for presentation at the Geologic Society of America 2017 annual meeting in Seattle, WA and the Minnesota GIS/LIS Consortium 2017 annual meeting in Bemidji, MN
