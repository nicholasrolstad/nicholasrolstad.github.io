Big Bend Interactive

This an interactive visitor map of Big Bend National Park created using the ESRI Leaflet library and HTML5, CSS, JavaScript & ArcGIS Online The data used is from the NPS and was processed in ArcMAP and ultimately hosted on ArcGIS Online as feature layers. A Python script is used on the server-side to scrape data concerning campsite availability from the web. Clicking on the screenshots below will take you to the live application. This demo application is limited in function for security/maintanence reasons (campsites status and condition reports are not actively being updated).
